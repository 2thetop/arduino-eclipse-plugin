Arduino Eclipse Plugin

This website is about a free Eclipse plugin to make development of big Arduino projects easier. All you need, is the Arduino IDE, Eclipse and this plugin. The Arduino IDE alternative is completely free and open source.

Download

Home

V2.3 is out.

The nightly build is in an experimental state though, use it only if you feel confident.

Features
  • Windows, Linux and OSX support
  • 64bit ready
  • Import of Libraries and other source code
  • Integrated serial monitor + scope
  • Multiple hardwares (version 2)
  • Support for the HalfKay boot loader (version 2)
  • Changing the COM port
  • Uploading to the board
  • Real time build
  • User selected build.
  • Simple install (Eclipse included in Download)
