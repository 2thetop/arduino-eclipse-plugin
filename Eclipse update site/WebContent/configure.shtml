This page contains Detailed instructions on how to configure the Arduino Eclipse plugin.
This page assumes you have installed the plugin correctly.

Detailed installation instructions

1) Start eclipse that contains the plugin. 1.1 ) open a workspace of your choice (no space in the name!!)
2.1) open the preferences (windows -> preferences)
2.2) open the section Arduino->Arduino (in V1) Arduino (in V2). Fill it in like below (using the location of the Arduino IDE of your choice)
Add the location of your private libraries.

The newer version no longer uses RXTX so there is no longer a "test serial dll" button. If the button is there click it.
V2 does not contain the option "use Arduino IDE tools in Eclipse".
V2.2 fills in mosts fields automatically and adds a "private hardware path" folder.
2.3) select OK to save the preferences

Congratulations: You have now successfully installed and configured the plugin.
